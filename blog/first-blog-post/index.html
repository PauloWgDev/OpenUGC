Well... We finally have more complete and organize documentation. It stills needs to be polished in many aspects, but should be enough for the first users of OpenUGC to get started.

Also, thanks to the guy who made the issue for a Demo/Tutorial. That motivated me to actually make this for once.

Future Plans​

We plan to soon publish U3GC on the Unity asset store. U3GC is the C# library for quick implementation of OpenUGC on unity.

Once we publish that we will also add the documentation on how to use the library on here. So all the information for the whole pipeline in on one place.

We plan to launch U3GC as a paid asset so if you think this API for UGC could be useful for you. And you want to quickly implement it on Unity, you could help us by getting the U3GC asset.

Note: So far we are calling the unity asset U3GC because it stands for "Unofficial Unity User Generated Content" but we might change this name in the future.
